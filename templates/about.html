About ProximaScan

ProximaScan is an advanced Face Recognition Attendance System built with Flask, OpenCV, and SQLite. It allows organizations to automate employee attendance with secure and reliable biometric authentication.

Features include:

  • Real-time face authentication with live webcam feed.
  • Attendance tracking with in/out timestamps.
  • Employee registration and management.
  • Export attendance records in CSV format.
  • Responsive dashboard for easy monitoring.

This system ensures accuracy, eliminates manual errors, and saves time in managing attendance records.
